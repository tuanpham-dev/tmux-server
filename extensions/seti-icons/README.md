# Seti Icons

The default file-icon theme: VS Code's own Seti icon set, giving the FILES tree colored icons per file type/language and folder state.

## Contributes

- **Icon theme "Seti":** font-glyph icons for common languages and file types, plus expanded/collapsed folder icons.

## Source

Extracted from VS Code's bundled `theme-seti` extension, MIT-licensed — see `LICENSE.txt`.

## Notes

Bundled with tmux-server as the default icon theme; pick a different one from Settings → UI → Icon theme (once another icon-theme extension is installed), or set it to "None" for blank spacer icons.
