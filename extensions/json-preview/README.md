# JSON/YAML Preview

A collapsible tree preview for `.json` and `.yaml`/`.yml` files, instead of raw text.

## Contributes

- **File viewer:** `.json` files render via [react-json-view-lite](https://github.com/AnyRoad/react-json-view-lite); `.yaml`/`.yml` files are parsed with the [yaml](https://github.com/eemeli/yaml) package and rendered the same way.

## Notes

Bundled with tmux-server — disable it from the Extensions view to fall back to plain-text editing for JSON/YAML files.
