# CSV Preview

An editable spreadsheet-style grid for `.csv`/`.tsv` files, opened as a preview tab instead of raw text.

## Contributes

- **File viewer:** `.csv`/`.tsv` files open as an editable grid (powered by [PapaParse](https://www.papaparse.com/)) — add/remove rows and columns, edit cells inline, and save back to the file.

## Notes

Bundled with tmux-server — disable it from the Extensions view to fall back to plain-text editing for CSV/TSV files.
