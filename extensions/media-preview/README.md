# Media Preview

An audio/video viewer tab with standard playback controls, opened instead of trying to render media files as text.

## Contributes

- **File viewer:** common audio (`.mp3`, `.wav`, `.ogg`, …) and video (`.mp4`, `.webm`, …) formats open as a native playback preview tab.

## Notes

Bundled with tmux-server — disable it from the Extensions view to fall back to downloading media files instead of previewing them.
