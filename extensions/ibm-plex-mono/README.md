# IBM Plex Mono

The default terminal font group: IBM's open-source monospace typeface, plus a Nerd Font-style symbols fallback.

## Contributes

- **Font group "IBM Plex Mono":** regular, italic, medium, and bold weights, with core latin + latin-ext coverage (English plus accented European text).
- **Perplexed:** VS Code's own supplementary glyph fallback font, riding along in the same selectable group so picking "IBM Plex Mono" in Settings → Terminal writes both families into the font stack at once.

## Notes

Bundled with tmux-server as the default terminal font; pick a different font group from Settings → Terminal → Font family.
